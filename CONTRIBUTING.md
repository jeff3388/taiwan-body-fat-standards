# Contributing to taiwan-body-fat-standards

感謝您對本資料庫的興趣。本文件說明如何貢獻及哪些貢獻不被接受。

## 資料來源要求

所有貢獻數據必須來自以下可信來源之一：

- 衛福部國民健康署官方標準（健康體位定義）
- ACSM（美國運動醫學會）指引
- WHO 相關報告和諮詢意見
- 已發表的同行評審期刊論文（需附 PMID）
- USDA FoodData Central 或衛福部食藥署 TFND（食物相關數據）

## 如何提交

1. Fork 此 repository
2. 新增或修改 JSON 資料
3. 確保每筆資料包含 `source` 欄位（含 PMID 或官方機構連結）
4. 更新 `_meta.last_updated`（YYYY-MM-DD）
5. 提交 Pull Request 並說明修改理由

## 接受的貢獻

- 新增其他國際認可的體脂肪標準（需標明出處）
- 更新測量方法準確度數據（需有新的驗證研究）
- 補充台灣在地測量資源（醫院、健身房等）
- 新增內臟脂肪相關的台灣本土研究數據
- 修正既有數據的錯誤

## 不接受的貢獻

- **特定體脂計品牌推薦**：商業利益衝突
- **偽科學測量方法**（如生物能量測試、虹膜診斷等）：缺乏科學依據
- **極端體脂率目標建議**（如男性 5%、女性 10%）：對非運動員有健康風險
- **無來源數據**：任何未附 PMID 或官方機構來源的健康聲明
- **SEO 目的的重複描述**：關鍵字填充

## 亞洲人體脂肪標準說明

本資料庫優先採用台灣衛福部及 WHO 亞洲諮詢意見，原因說明如下：

亞洲人與歐美人在相同 BMI 下體脂肪比例不同（PMID: 12022029），因此：
- 台灣肥胖 BMI 切點為 27（非 WHO 全球標準 30）
- 體脂肪健康範圍應使用適合亞洲族群的標準

貢獻數據時請標明所採用的標準（`standard` 欄位），勿混用不同標準。
