# Taiwan Body Fat Standards（台灣體脂肪標準資料庫）

台灣體脂肪率健康標準的結構化資料庫。收錄依性別、年齡分類的體脂肪率健康範圍（台灣衛福部 vs ACSM 標準對照）、各種測量方法的準確度評估，以及內臟脂肪與代謝風險的量化指標。

## 為什麼台灣需要獨立的體脂肪標準

台灣採用**亞洲人適用的體脂肪切點**（衛福部，2014），與歐美 ACSM/WHO 標準不同，原因是：

> 同樣 BMI 25 的亞洲人，體脂肪比例比歐美族群高 **3–5 個百分點**，且內臟脂肪面積更大（Deurenberg et al., 2002）。

這意味著，用歐美標準判斷台灣人的體脂是否「正常」，會系統性地**低估**代謝風險。

| 族群 | BMI 25 的平均體脂率 | 代謝症候群風險 |
|------|-------------------|--------------|
| 歐美白人 | 約 22–24%（男）/ 31–33%（女） | 基準 |
| 東亞（台灣/日本/韓國） | 約 25–27%（男）/ 34–36%（女） | +30–40% |

## BMI vs 體脂率：哪個更準確？

BMI 的根本問題在於它**無法區分肌肉和脂肪**，也無法反映脂肪的分布位置。

| 情境 | BMI | 體脂率 | 真實代謝風險 |
|------|-----|--------|------------|
| 肌肉發達運動員 | 25–27（過重） | 12–18%（健康） | 低 |
| 泡芙人（TOFI） | 22–23（正常） | 28–35%（肥胖） | 高 |
| 一般體型亞洲人 | 24（臨界） | 25–30%（過重） | 中至高 |

台灣研究顯示：以 BMI 24 為切點，**漏診約 30% 的代謝異常個案**（體脂率過高但 BMI 正常）。

## 各種體脂肪測量方法比較

| 方法 | 準確度 | 成本 | 說明 |
|------|--------|------|------|
| DEXA（雙能量 X 光） | ⭐⭐⭐⭐⭐ | 高（約 NT$2,000–5,000） | 黃金標準，可分區域（軀幹/四肢/內臟） |
| 水中秤重（密度法） | ⭐⭐⭐⭐⭐ | 高（研究設備） | 學術黃金標準，不便攜帶 |
| BIA（生物電阻抗） | ⭐⭐⭐ | 低（家用體脂機） | 受水合狀態影響大，同日不同時間誤差 3–5% |
| 皮脂夾（Skinfold） | ⭐⭐⭐ | 低（需訓練操作者） | 技術依賴性高，肥胖者誤差大 |
| 美國海軍圍度法 | ⭐⭐ | 免費（捲尺即可） | 用腰圍/頸圍/身高估算，誤差 3–8%，無法反映內臟脂肪 |

**建議：** 無論選用哪種方法，重要的是**固定同一方法追蹤趨勢**，而不是比較不同方法的絕對數值。

## 內臟脂肪的重要性

體脂率只告訴你**總體脂量**，但**脂肪分布**同樣關鍵。內臟脂肪（腹腔深層脂肪）比皮下脂肪代謝活躍，對代謝風險的影響遠高於總體脂率。

**內臟脂肪的臨床指標：**

| 指標 | 正常切點（亞洲人） | 說明 |
|------|----------------|------|
| 腰圍 | 男 < 90cm，女 < 80cm | 台灣衛福部標準，亞洲切點比歐美嚴格 |
| 腰臀比（WHR） | 男 < 0.90，女 < 0.85 | 脂肪分布指標 |
| 腰高比（WHtR） | < 0.5 | 研究顯示比 BMI 更能預測心血管風險 |
| ABSI（身體形狀指數） | 越低越好 | 新型複合指標，考慮腰圍/BMI/身高 |

## 相關工具（Interactive Tools）

本資料庫的體脂肪標準數據，已整合至 [MetaboLab](https://metabolab.tw) 的互動工具中：

- 📐 **[體脂率計算機](https://metabolab.tw/calculators/body-fat)**：使用美國海軍圍度法（腰圍 + 頸圍 + 身高）估算體脂肪率，對照台灣衛福部分年齡層健康標準。
- 📏 **[腰圍風險計算機](https://metabolab.tw/calculators/waist-risk)**：評估腰圍（內臟脂肪的代理指標）是否超出台灣衛福部切點（男 90cm / 女 80cm），並計算對應的代謝症候群風險。
- 📊 **[TDEE 計算機](https://metabolab.tw/calculators/tdee)**：在了解目前體脂率後，設定合理的熱量赤字計畫——體脂率目標比體重目標更能反映真實的身體組成改善。
- 🏠 **[BMI 計算機](https://metabolab.tw/calculators/bmi)**：對照 BMI 與體脂率，理解為什麼「BMI 正常不等於體脂健康」，尤其對台灣的泡芙人（TOFI）族群。
